# Support

CANARY is a research prototype. Support is provided on a best-effort basis.

## How to get help

| Channel | When to use |
|---------|-------------|
| [GitHub Issues — bug report](https://github.com/timmybx/canary/issues/new?template=bug_report.md) | Reproducible bugs or unexpected behavior |
| [GitHub Issues — feature request](https://github.com/timmybx/canary/issues/new?template=feature_request.md) | Suggestions or enhancements |
| [GitHub Discussions](https://github.com/timmybx/canary/discussions) | General questions, usage help, or open-ended discussion |

## Security issues

Please **do not** open a public issue for security vulnerabilities.
Follow [SECURITY.md](SECURITY.md) and report via GitHub Security Advisories or email.

## What to include

When opening a bug report, please include:

- Your OS and Python/Docker version
- The exact command you ran
- The full error message or unexpected output
- Steps to reproduce
